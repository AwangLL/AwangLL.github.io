## 注释

```java
// 单行注释

/* 
	多行注释 
*/

/**
 * JavaDoc
 */
```

### javaDoc

常用参数信息

- author: 作者
- deprecated: 标识过期的类或成员
- exception: 标识抛出的异常
- param: 标识参数名
- return: 标识返回值情况
- see: 标识指定参数的内容
- serial: 标识序列化属性
- version: 标识版本号
- since: 标识指明需要最早使用的jdk版本
- throws: 标识异常抛出问题

生成javaDoc，在命令行中输入

```bash
javadoc -encoding UTF-8 -charset UTF-8 javaDocTest.java
```
