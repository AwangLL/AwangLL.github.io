# 运算符

基本运算符：`+ - * / % -- ++`
幅值运算符：`=`
关系运算符：`> < >= <= == != instanceof`
逻辑运算符：`&& || !`
位运算符：`& | ^ ~ >> << >>>`
条件运算符：`?:`
扩展运算符：`+= -= *= /=`

## 优先级

|优先级|运算符|结合性|
|---|---|---|
|1|()、\[]、{}|从左向右|
|2|!、+、-、~、++、--|从右向左|
|3|\*、/、%|从左向右|
|4|\+、\-|从左向右|
|5|<<、>>、>>>|从左向右|
|6|<、<=、>、>=、instanceof|从左向右|
|7|\==、!=|从左向右|
|8|&|从左向右|
|9|^|从左向右|
|10|\||从左向右|
|11|&&|从左向右|
|12|\|\||从左向右|
|13|?:|从右向左|
|14|=、+=、-=、*=、/=、&=、\|=、^=、~=、<<=、>>=、>>>=|从右向左|
