# 标识符和关键字

## 标识符

由字母、数字、\_、$组成，但标识符不能以数字开头，不能是Java中的关键字。

### 标识符书写规范

- 包名所有字母一律小写。
- 类名和接口名每个单词的首字母都大写。
- 常量所有字母都大写，单词之间用下划线连接。
- 变量名和方法名的第一个单词首字母小写，从第二个单词开始每个单词首字母大写。
